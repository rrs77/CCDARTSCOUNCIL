# Partner logos

White wordmark SVGs for the login **Supported by** strip on the dark green (`#002D24`) hero.

## Replacing with official assets

1. Export each partner’s **white** or **reversed** logo from their brand guidelines (PNG or SVG).
2. Save using the same filename (e.g. `royal-opera-house.svg`).
3. Keep a wide aspect ratio; height is controlled in the UI (~28–36px).
4. If your file is dark-on-transparent, add `invert: true` for that entry in `src/config/partnerLogos.ts` and use the `partner-logo--invert` class (see `PartnerLogoStrip.tsx`).

Do not redraw or alter official logos beyond scaling and approved colour reversals.

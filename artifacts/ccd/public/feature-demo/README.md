# CCDesigner feature demo (external organisations)

Shareable walkthrough of the **working CCDesigner prototype** — mostly live UI, with a few full-frame title slides.

## Slide colour palette (fixed 1920×1080 video — not responsive)

| Token | Hex | Use |
|-------|-----|-----|
| `--bg-deep` | `#05231e` | Deepest background |
| `--bg-forest` | `#002D24` | Primary forest field |
| `--text` | `#FFFFFF` | Headlines |
| `--text-body` | `#F2F7F4` | Body copy (high contrast) |
| `--accent` | `#B6FF7E` | Eyebrows / underlines only |

## Linear pathway (full-length — do not shorten)

1. **Logo**  
2. **Context A** — Performing arts under pressure (schools & FE)  
3. **Context B** — One place to plan and connect  
4. **Disclaimer**  
5. **Live login** — connection hero only (no lightning on page)  
6. **Ideas slide** — lightning / melting pot  
7. **Action** — Let's see the app in action  
8. **Explore** — Half-Term Designer home  
9. **LSO hub** — click arts org → choose **How to Build an Orchestra** → **Add to CCDesigner**  
10. **Activity Library** — same LSO topic activities visible  
11. **Lesson Builder → Lesson Library** — create a named lesson from those activities, save, then show that same lesson in Lesson Library before assign  
12. **Term overview** — lesson from Library on Half-Term Designer  
13. **Key dates**  
14. **Settings**  
15. **Create** (own activity/lesson → Library)  
16. **Calendar week**  
17. **Paid** (WTD / iCompose · 10–20%)  
18. **Organisations**  
19. **Closing** — contacts (not thank you)  

## Audio
